# clever-frontend

## Como rodar

Primeiro clone o projeto:

```bash
> git clone https://github.com/raphaelvaz/clever-frontend.git
```
Depois entre na pasta via terminal:

```bash
> cd /clever-frontend
```
Instale todas as dependências:

```bash
> yarn
```
Starte o projeto:

```bash
> yarn start
```

Acesse o endereço:
```bash
> http://localhost:3000
```
